# TruckGuard

Aplicação comunitária para alertas rodoviários em tempo real para motoristas profissionais.

## Funcionalidades

- Alertas comunitários
- Radar em tempo real
- Google Maps
- Waze
- Alertas por proximidade
- Votação da comunidade
- Supabase Realtime
- Voto único por dispositivo
- Expiração automática de alertas

## Tecnologias

- React Native
- Expo
- TypeScript
- Supabase
- AsyncStorage
- Leaflet

## Instalação

```bash
npm install
npx expo start
```
